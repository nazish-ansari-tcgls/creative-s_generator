
# 🎨 Creative Generator

![Python](https://img.shields.io/badge/python-3.9%2B-blue.svg)
![AI](https://img.shields.io/badge/AI-Generative%20Creative-purple.svg)

**Creative Generator** is an AI-powered creative automation framework built using a **multi-agent architecture** to generate high-quality marketing creatives at scale.

---

## 🚀 Features
- 🧠 Multi-agent creative orchestration
- 🎯 Prompt generation & optimization
- 🎨 Brand-aware visual styling
- 🖼️ Image understanding & enhancement
- ✅ Automated QA validation
- 🧩 Modular & extensible design

---

## 🏗️ Architecture
Each agent performs a specialized role and contributes to a refined final creative output.

```
Input → Category → Prompt → Visual → Styling → Messaging → QA → Synthesis → Output
```
